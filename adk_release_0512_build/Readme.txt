AndroidADK_2_3_4_or_more.apk is just a rebuild of https://dl-ssl.google.com/android/adk/adk_release_0512.zip
It is build with Google APIs Level 10 add-on Library which includes the Open Accessory library for 2.3.4 devices that support accessory mode.
It is compatible with Android 2.3.4 or more.
Nota:
It is a big package because of resource(pictures) and especially joystick_background.png which is more than 160KB and background_holo_dark.png which is up to 1.8MB,
it support multiple screen (drawable-hdpi, drawable-mdpi, drawable-nodpi, drawable-xlarge-nodpi).
